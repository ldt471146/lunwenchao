# 基于贝叶斯网络的危险化学品道路运输风险评估

## 论文信息

- **题目**: 基于贝叶斯网络的危险化学品道路运输风险评估
- **作者**: 蒋维超
- **学号**: 202217150148
- **专业**: 物流工程(校企合作本科)
- **学院**: 物流学院
- **学校**: 临沂大学
- **完成时间**: 2025年5月

## 摘要

危险化学品道路运输作为现代物流体系的重要组成部分，其安全风险评估对保障公共安全具有重要意义。本研究针对危险化学品道路运输系统风险因素众多、相互作用关系复杂、不确定性强的特点，构建了基于贝叶斯网络的风险评估模型。

## 研究内容

1. **理论框架**: 建立"人-车-路-环境-管理"五维度风险分析框架
2. **数据分析**: 收集并分析218起历史事故数据
3. **模型构建**: 构建包含18个节点的贝叶斯网络模型
4. **风险评估**: 进行正向推理、反向诊断和敏感性分析
5. **管控建议**: 提出针对性的风险管控措施

## 主要成果

- 模型预测准确率: **86.4%**
- AUC值: **0.892**
- 识别关键风险因素: 车辆技术状况、驾驶员疲劳、安全管理
- 生成专业图表: **9张**
- 参考文献: **30篇** (符合GB/T7714-2015标准)
